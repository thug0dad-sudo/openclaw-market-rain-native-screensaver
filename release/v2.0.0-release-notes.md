# OpenClaw Market Rain Native Screensaver v2.0.0

Native macOS Screen Saver release with live quote support and ticker settings.

## Highlights

- Native macOS .saver bundle
- Live quote tokens from the Market Rain API
- Ticker settings panel through Screen Saver Options/Settings
- Default symbols: NVDA,BTC,ETH,RKLB,LUNR,ASTS,PLTR,SPY,QQQ
- Matrix-style falling ticker rain
- OLED-friendly black background
- MIT free-use license

## Install

1. Download OpenClawMarketRainNative-v2.0.0.saver.zip.
2. Extract the zip.
3. Move the .saver bundle to ~/Library/Screen Savers/.
4. Open macOS System Settings.
5. Go to Screen Saver.
6. Select OpenClaw Market Rain 2.0.0.
7. Use Options/Settings to configure tickers.
